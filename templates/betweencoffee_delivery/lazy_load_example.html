🖼️ 圖片懶加載演示

以下圖片使用懶加載技術，只有當滾動到可視區域時才會加載。

💻 使用方法

方法1: 使用data屬性（推薦）

方法2: 使用JavaScript API

// 手動加載特定圖片 const image = document.querySelector('#my-image'); window.LazyLoad.load(image); // 加載所有可見圖片 window.LazyLoad.loadAllVisible(); // 檢查WebP支持 window.LazyLoad.checkWebPSupport().then(supported => { console.log('WebP supported:', supported); }); // 監聽加載完成事件 document.addEventListener('lazyloaded', (event) => { console.log('圖片加載完成:', event.detail.element); });

🎯 功能特性

  • ✅ Intersection Observer API - 高效檢測元素可見性
  • ✅ WebP自動檢測 - 自動使用WebP格式（如果瀏覽器支持）
  • ✅ 骨架屏效果 - 圖片加載時顯示佔位符
  • ✅ 錯誤重試機制 - 網絡錯誤時自動重試
  • ✅ 動態內容支持 - 自動處理動態添加的圖片
  • ✅ 性能監控 - 實時統計加載狀態
  • ✅ 移動端優化 - 響應式設計，觸控友好
  • ✅ 無依賴 - 純JavaScript實現，無需第三方庫
