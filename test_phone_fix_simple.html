電話號碼格式化修復測試

測試 CommonUtils.formatPhoneNumber() 函數

// 測試函數邏輯 function formatPhoneNumber(phone) { if (!phone) return ''; // 移除所有空格 let cleaned = phone.toString().replace(/\s+/g, ''); // 移除香港地區碼 (+852) cleaned = cleaned.replace(/^\+852/, ''); cleaned = cleaned.replace(/^852/, ''); return cleaned; }

實際渲染器電話顯示測試

// 模擬渲染器中的電話顯示邏輯 const order = { id: 12345, name: "測試顧客", phone: "+852 1234 5678", pickup_code: "A123" }; // 渲染器中的代碼 const phone = window.CommonUtils ? window.CommonUtils.formatPhoneNumber(order.phone || '') : (order.phone || ''); // 顯示結果 const displayHTML = `電話: ${phone}`;

問題診斷

如果電話仍然顯示 "+852"，可能的原因：

  1. 瀏覽器緩存 - 請清除瀏覽器緩存或使用 Ctrl+F5 強制刷新
  2. JavaScript 文件未正確加載 - 檢查瀏覽器控制台是否有錯誤
  3. CommonUtils 未正確初始化 - 檢查 CommonUtils 是否已註冊到 window 對象
  4. 電話數據格式問題 - 檢查後端返回的電話數據格式
