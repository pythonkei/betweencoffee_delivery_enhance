咖啡師顯示功能測試

測試目的

驗證系統是否正確顯示製作中訂單的咖啡師信息，並確保後端API正確記錄咖啡師名稱。

測試步驟

步驟 1: 檢查後端API

驗證後端API是否從 request.user 獲取咖啡師名稱。

// eshop/views/queue_views.py - start_preparation_api 函數 @require_POST @login_required @staff_member_required def start_preparation_api(request, order_id): try: barista_name = request.user.get_full_name() or request.user.username result = OrderStatusManager.mark_as_preparing_manually( order_id=order_id, barista_name=barista_name ) ...
後端API已正確配置：從 request.user 獲取咖啡師名稱

步驟 2: 檢查數據處理器

驗證 OrderItemProcessor 是否正確處理咖啡師信息。

# eshop/utils/order_item_processor.py - _prepare_queue_info 方法 def _prepare_queue_info(queue_item, now=None, hk_tz=None): queue_info = { 'queue_item_id': queue_item.id, 'position': queue_item.position, 'status': queue_item.status, 'preparation_time_minutes': queue_item.preparation_time_minutes, } # 添加咖啡師信息 if hasattr(queue_item, 'barista') and queue_item.barista: queue_info['barista'] = queue_item.barista elif hasattr(queue_item.order, 'barista') and queue_item.order.barista: queue_info['barista'] = queue_item.order.barista else: queue_info['barista'] = '未分配' ...
數據處理器已正確配置：包含咖啡師信息處理邏輯

步驟 3: 檢查前端顯示

驗證前端JavaScript是否正確顯示咖啡師信息。

// static/js/staff-order-management/queue-manager.js - updatePreparingQueueImmediately 方法 row.innerHTML = ` #${orderId} ${order.pickup_code || ''}

使用說明

  1. 員工登錄系統後，點擊"開始製作"按鈕
  2. 系統自動記錄當前用戶為咖啡師
  3. 製作中訂單表格會顯示咖啡師姓名
  4. 如果訂單沒有分配咖啡師，會顯示"未分配"
  5. 咖啡師信息有助於追蹤責任和協調工作

下一步操作

要測試實際功能，請執行以下步驟：

  1. 啟動Django開發伺服器：python manage.py runserver
  2. 登錄員工管理界面
  3. 找到等待中的訂單，點擊"開始製作"
  4. 檢查製作中訂單表格是否顯示您的用戶名作為咖啡師
  5. 驗證多個員工操作時，每個訂單顯示正確的咖啡師
注意：需要實際登錄測試才能完全驗證功能
