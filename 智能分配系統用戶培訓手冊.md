## 📋 培訓概述

本手冊旨在幫助咖啡店員工和管理員快速掌握智能分配系統的使用方法。智能分配系統是一個基於員工工作負載追蹤的智能訂單分配系統，能夠：

1. **智能分配訂單** - 自動將訂單分配給最合適的員工
2. **追蹤工作負載** - 實時監控每個員工的工作狀態
3. **優化製作流程** - 最大化並行製作效率
4. **提供智能建議** - 為管理決策提供數據支持

## 📚 培訓內容

### 第一部分：系統介紹（30分鐘）

#### 1.1 什麼是智能分配系統？
- **系統目的**: 優化訂單分配，提高製作效率
- **核心功能**: 
  - 員工工作負載追蹤
  - 智能訂單分配
  - 並行製作優化
  - 性能監控和分析
- **預期效益**:
  - 員工利用率提升20-30%
  - 訂單處理速度提升15-25%
  - 工作負載更均衡
  - 錯誤率降低

#### 1.2 系統界面介紹
- **員工界面**: 工作負載顯示、訂單列表、操作按鈕
- **管理界面**: 系統監控、數據分析、配置管理
- **移動端支持**: 響應式設計，支持手機和平板

#### 1.3 基本概念解釋
- **工作負載**: 員工當前的工作量
- **並行容量**: 員工同時可以處理的訂單數
- **效率因子**: 員工的製作效率評分
- **智能建議**: 系統提供的優化建議

### 第二部分：員工操作培訓（60分鐘）

#### 2.1 登入系統
1. **訪問員工界面**: 打開瀏覽器，訪問員工管理頁面
2. **輸入憑證**: 使用分配的用戶名和密碼登入
3. **確認身份**: 系統會顯示您的姓名和當前狀態

#### 2.2 查看工作負載
1. **工作負載面板**: 在界面左側查看您的工作負載
2. **當前訂單**: 查看您正在處理的訂單列表
3. **可用容量**: 查看您還可以接多少新訂單
4. **預計時間**: 查看每個訂單的預計完成時間

#### 2.3 處理訂單
1. **接收新訂單**: 系統會自動為您分配新訂單
2. **開始製作**: 點擊訂單上的"開始製作"按鈕
3. **更新狀態**: 在製作過程中更新訂單狀態
4. **完成訂單**: 製作完成後點擊"完成"按鈕

#### 2.4 查看智能建議
1. **建議面板**: 在界面右側查看系統建議
2. **分配建議**: 查看系統推薦的訂單分配方案
3. **優化建議**: 查看如何提高效率的建議
4. **時間預估**: 查看準確的預計製作時間

#### 2.5 常見操作
1. **暫停工作**: 需要休息時可以暫停接收新訂單
2. **恢復工作**: 休息結束後恢復接收新訂單
3. **查看歷史**: 查看您處理過的訂單歷史
4. **報告問題**: 遇到問題時使用報告功能

### 第三部分：管理員操作培訓（90分鐘）

#### 3.3 數據分析

3. **趨勢預測**: 預測訂單量和資源需求

### 2. 常見問題解答
```markdown
# 常見問題解答

## Q: 無法登入系統怎麼辦？
A: 1. 檢查網絡連接
   2. 確認用戶名和密碼正確
   3. 聯繫管理員重置密碼

## Q: 系統沒有分配新訂單？
A: 1. 檢查您的工作狀態是否為"在崗"
   2. 檢查您的可用容量
   3. 檢查系統是否有待分配訂單

## Q: 預計時間不準確？
A: 1. 檢查訂單的咖啡數量
   2. 檢查您的效率因子設置
   3. 報告問題給管理員

## Q: 系統運行緩慢？
A: 1. 檢查網絡連接
   2. 清理瀏覽器緩存
   3. 聯繫技術支持
```
