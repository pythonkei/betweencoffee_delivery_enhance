  • 問題：JavaScript 調用了錯誤的 API 路徑

  • 正確路徑：/eshop/order/api/order-status/{order_id}/

    • 
            

            
            
修復內容

            
    
                
  • 修復文件：static/js/order_payment_confirmation.js
    •

    •

  • 修復後：const response = await fetch(`/eshop/order/api/order-status/${this.orderId}/`);
