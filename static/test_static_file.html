🔍 測試方法

方法 1：直接訪問靜態文件

在瀏覽器中打開：

http://localhost:8081/static/js/staff-order-management/order-detail.js

方法 2：檢查 Network 標籤

  1. 按 F12 打開開發者工具
  2. 切換到 Network 標籤
  3. 刷新頁面
  4. 過濾 "js" 查看 JavaScript 文件
  5. 檢查 order-detail.js 的狀態碼

方法 3：清除瀏覽器緩存

按 Ctrl+Shift+Delete (Windows) 或 Cmd+Shift+Delete (Mac) 清除緩存

📝 常見問題解決方案

問題 1：文件 404 錯誤

解決方案： 檢查 Django 靜態文件配置

# settings.py STATIC_URL = '/static/' STATICFILES_DIRS = [os.path.join(BASE_DIR, 'static')]

問題 2：瀏覽器緩存

解決方案： 強制刷新頁面

  • Windows: Ctrl+F5
  • Mac: Cmd+Shift+R
  • 或使用開發者工具中的 "Disable cache" 選項
